Roommate Sharing App

App build for my girlfriends appartment to help them track shared items.

Build fully with SwiftUI and Firebase backend.

Beta Version 1.0
4 users (the 4 roommates in the appartment.


New feautures
-single room (name from request of girlfriend)
-list of items shared
-ability to change each item between 3 states: have, low, need
-add item (button)
-delete item (swipe action)
-filter items based on states
